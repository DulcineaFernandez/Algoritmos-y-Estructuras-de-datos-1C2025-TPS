# Algoritmos y Estructuras de Datos - Trabajos Prácticos

Este directorio contiene los trabajos prácticos realizados para la materia Algoritmos y Estructuras de Datos. A continuación se describe brevemente el contenido de cada uno.

## [TP0 - Algoritmos y Estructuras de Datos](./tp0)
**Manejo de Archivos y Ordenamiento**
Programa introductorio que lee números enteros de dos archivos, compara los arreglos resultantes y ordena el "mayor" utilizando el algoritmo de Selección (Selection Sort).
*   **Lenguaje**: Go

## [TP1 - Notación Infija a Posfija](./tp1)
**Pilas y Colas (Shunting-yard)**
Implementación de un convertidor de expresiones matemáticas de notación infija a posfija. Soporta operaciones aritméticas básicas y paréntesis, utilizando el algoritmo Shunting-yard y TDA Pila/Cola.
*   **Lenguaje**: Go

## [TP2 - Sistema de Gestión de Vuelos (Algueiza)](./tp2)
**Estructuras de Datos Avanzadas (Hash, Heap, Árboles)**
Sistema para administrar un tablero de vuelos. Permite cargar datos desde archivos CSV y realizar consultas eficientes como ver el tablero ordenado por fecha, buscar el siguiente vuelo, ver vuelos prioritarios y borrar vuelos.
*   **Lenguaje**: Go

## [TP3 - Sistema de Consultas de Vuelos (FlyCombi)](./tp3)
**Grafos y Algoritmos de Caminos Mínimos**
Sistema de consultas para una aerolínea utilizando grafos. Permite encontrar caminos mínimos (por tiempo o precio), caminos con menos escalas, calcular centralidad de aeropuertos, generar redes mínimas (MST) y exportar rutas a KML.
*   **Lenguaje**: Python
