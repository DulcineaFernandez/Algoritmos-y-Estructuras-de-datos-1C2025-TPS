# TP0 - Algorithms and Data Structures

- File habilities
- Array manipulation
- Selection sort algorithm

## Description

The program (`main.go`) reads two files with integer numbers (`archivo1.in` and `archivo2.in`). It puts the numbers in two integer arrays and compares them to determine which one is "larger". Finally, it sorts the larger array using Selection Sort and prints it. 

## Requirements

- Two input files named `archivo1.in` and `archivo2.in` in the same directory as `main.go`.
